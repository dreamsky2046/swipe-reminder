# 刷卡提醒 App

一个帮助你追踪银行卡刷卡次数的 React Native (Expo) 应用。

## 功能

- 添加/编辑/删除银行卡信息
- 追踪每张卡的已刷卡次数
- 支持启用/禁用银行卡
- 数据本地持久化存储

## 字段说明

- 银行：银行名称
- 卡号：银行卡尾号
- 有效期：卡片有效期
- 识别码：手动录入的数字
- 重置日期：刷卡次数重置日期
- 次数要求：每月/每年需要的刷卡次数（两位数）
- 是否启用：是/否

## 开发

```bash
npm install
npm start
```

## 构建 APK

需要配置 EAS Build 或 GitHub Actions。详见项目根目录的 GitHub Actions workflow 配置。
